# yangs-mirai-bot

基于mirai-api-http实现的python机器人

## 功能

- [x] 基础命令系统
- [x] OpenAI对话
- [x] 状态查询
- [x] 一言
- [x] Lolicon API
- [x] B站视频信息查询
- [x] 计划任务
- [x] 词云生成
- [ ] 正则匹配事件


## 使用

1. 安装依赖

```shell
pipenv install
```

2. 修改配置文件

```shell
cp config.yml.template config.yml
your_editor config.yml
```

3. 运行

```shell
pipenv run python main.py
```
